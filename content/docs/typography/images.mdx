## Image effects

Use image effects such as grayscale or blur to change the appearances of the image when being hovered.

### Grayscale

Use the filter option and apply a grayscale to the image element using the grayscale class.

### Blur

Apply a blur by using the blurLevel option which takes string xs, sm, md, lg from Tailwind CSS to an image component.

### Alignment

Align the image component to the left, center or right side of the document page using margin styles.

### Left

By default, the image component will be aligned to the left side of the page.

### Center

Horizontally align the image to the center of the page using the mx-auto class.

### Right

Use the ml-auto class to align the image to the right side of the page.

## Sizes

Set the size of the image using the and or utility classes from Tailwind CSS to set the width and height of the element.

### Small

Use the size="sm" to set a small size of the image.

### Medium

Use the size="md" to set a medium size of the image.

### Large

Use the size="lg"s to set a large size of the image.

### Full width

Use the size="full" to set the full width of the image as long as it doesn’t become larger than the original source.
